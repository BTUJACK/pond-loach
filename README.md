## 安装

这个项目使用[Python](https://www.python.org/) [Git](https://git-scm.com/) 请确保你本地安装了它们。

```shell
$ git clone https://github.com/weiensong/Fortune500.git
```


## 运行
```sh
$ pip install -r requriements.txt

# 爬取
$ python3 ./crawler.py

# 作图
$ python3 ./analyze.py
```
